# LoveLetter 💕

A romantic webpage to express love to my dearest.

## 📖 Description

This is a beautiful love letter webpage designed for long-distance couples. It features:
- Romantic gradient background
- Floating heart animations
- Responsive design
- Elegant typography
- Deeply touching love messages

## 🎨 Features

- **Beautiful Design**: Gradient purple-pink background with floating hearts
- **Animations**: Heartbeat effects and floating heart particles
- **Responsive**: Works perfectly on both mobile and desktop
- **Expressive Content**: Deeply touching words for long-distance lovers

## 🚀 How to Use

Simply open `index.html` in your browser to view the love letter.

## 💝 Customization

You can customize the content by editing the following sections in `index.html`:
- Love messages
- Reasons for loving someone
- Personal promises

## 🌟 For Long-Distance Couples

This webpage is specially designed for couples in long-distance relationships, expressing:
- Distance can never separate true love
- The promise of a future together
- Deep思念 (missings) and love
